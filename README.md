# 🚀 Full Stack Portfolio Website

A modern and responsive portfolio website built using **React.js**, **Tailwind CSS**, and **FastAPI**. This project showcases my technical skills, featured projects, resume, and contact information through a clean and interactive user interface.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm run dev
```

### Run Backend

```bash
uvicorn main:app --reload
```
